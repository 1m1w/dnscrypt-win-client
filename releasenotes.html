Version 0.0.2 early preview testing

Updated: Wed, 25 April 2012

  • Minor bug fixes and a test for auto upgrade included. Version coming soon with early adopter bug fixes included.

Version 0.0.3 early preview testing

Updated: Thurs, 3 May 2012

  • Bug Fix: "OpenDNSInterface has encountered errors and needs to close" bug fixed. It was a settings file corruption issue.
  • Bug Fix: DNSCrypt fails over if encrypted traffic is dropped by some ISPs and Routers (fail over to insecure must be checked)
  • Reset DNS to default when user exits DNSCrypt via menu

Version 0.0.4 early preview testing

Updated: Mon, 7 May 2012

  • DNSCrypt now handles captive portals more cleanly. When "Use DNSCrypt" and "Fail over to insecure" are checked and a captive portal is encountered, the program fails open internally, but continues to attempt to turn on encryption as soon as it can. The old behavior simply turned off encryption until the user re-checked it.
  • "Fail over to insecure" is checked by default.
